## Simple Basket Calculator 
Ruby application with RSpec
### Dependencies:
- Ruby 3.3.0

### Installation:
- Clone poject
- Run bundler:

 ```shell
 $ bundle install
 ```

##### Tests:
To execute automation tests, run following commands:

```shell
 $ bundle exec rspec
```
